# Contributing
We welcome contributions aligned with our **Ethics & Consent**.

## Process
1. Fork + create a feature branch.
2. Ensure all code files include `SPDX-License-Identifier: AGPL-3.0-only`.
3. Run tests and linters; add/expand docs.
4. Confirm you’ve read **ETHICS.md**, **AUP.md**, and **TRADEMARKS.md**.
5. In your PR, paste the CLA sign‑off string from `CLA.md`.

## PR Checklist
- [ ] Ethics checked (RMRIΔ honoured; no hidden data flows)
- [ ] Unit tests & docs updated
- [ ] No private data in fixtures
- [ ] Glyph and brand usage compliant
